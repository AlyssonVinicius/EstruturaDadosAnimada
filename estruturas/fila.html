Uma fila é uma estrutura de dados linear que segue o princípio FIFO (First In, First Out), o que significa que o primeiro elemento inserido é o primeiro a ser removido. Pense nisso como uma fila em um supermercado, onde a pessoa que chegou primeiro é atendida primeiro.


 Características Principais da Fila:

 - Enfileirar (Enqueue): Adicionar um elemento ao final da fila.

 - Desenfileirar (Dequeue): Remover o elemento do início da fila.

 - Frente (Front): O primeiro elemento adicionado à fila.

 - Fundo: O último elemento adicionado à fila.

 - Tamanho: O número de elementos na fila.

 - Vazia (isEmpty): Verificar se a fila está vazia.

 - Cheia (isFull): Em algumas implementações, verifica se a fila está cheia (pode ser relevante em filas com tamanho fixo).


 Operações Básicas:

 - Enfileirar (Enqueue): Adiciona um elemento ao final da fila.

 - Desenfileirar (Dequeue): Remove o elemento do início da fila.

 - Frente (Front): Obtém o elemento no início da fila sem removê-lo.

 - Vazia (isEmpty): Verifica se a fila está vazia.

 - Cheia (isFull): Verifica se a fila está cheia (em algumas implementações). Implementação:


 A implementação de uma fila pode ser feita usando uma estrutura de dados como um array ou uma lista encadeada. Em linguagens de programação orientadas a objetos, é comum criar uma classe para representar uma fila, e as operações básicas podem ser métodos dessa classe.


 Aplicações de Filas:

 - Controle de Recursos: Filas são frequentemente usadas para controlar o acesso concorrente a recursos compartilhados, como impressoras ou CPUs.

 - BFS (Busca em Largura): Algoritmos de busca em largura em grafos usam filas para explorar níveis sucessivos de um grafo.

 - Agendamento de Tarefas: Filas são usadas para gerenciar tarefas em muitos sistemas, garantindo que a ordem de chegada seja respeitada.


 As filas são uma estrutura de dados fundamental em ciência da computação e são amplamente usadas em diversos contextos.
